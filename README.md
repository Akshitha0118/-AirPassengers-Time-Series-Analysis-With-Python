# ✈️ -AirPassengers-Time-Series-Analysis-With-Python

This project performs Time Series Analysis on the famous AirPassengers dataset to understand long-term growth trends and seasonal patterns in airline passenger traffic.

---

## 📊 Project Overview

The AirPassengers dataset contains monthly totals of international airline passengers from 1949 to 1960.  

The goal of this project is to:

- Perform data preprocessing
- Conduct exploratory data analysis (EDA)
- Visualize trends and seasonality
- Analyze growth patterns over time
- Understand basic time series concepts

---

## 🔍 Key Insights

- Clear upward growth trend in passenger traffic
- Strong seasonal patterns observed annually
- Increasing variation over time (multiplicative trend behavior)

---

## 🛠️ Technologies Used

- Monthly seasonality analysis
